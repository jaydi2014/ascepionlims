Installation/System Requirements

Marvin is Java based and so can run on all major operating systems.
Marvin products come in various packages for users with different needs:

Marvin Applets:

Marvin Applets are targeted at the Web Developer who builds chemical Internet/Intranet sites.
The package contains:
MarvinSketch:
Applet for editing and visualizing structures, queries and reactions on a web page.
MarvinView:
Applet for viewing structures, queries and reactions on a web page.
MarvinSpace:
Applet for 3D molecule visualization.

Note, that the applets are signed that allows the same feature set as the applications.

Marvin Applications:

Standalone Java applications (MarvinSketch, MarvinView and MarvinSpace)

System requirements for Marvin products are listed below:

Marvin Applets 

Java 2 enabled browser that supports 1.4 or later version of Java

Marvin Applications 

Java Runtime Environment (JRE) 1.4 or later.
