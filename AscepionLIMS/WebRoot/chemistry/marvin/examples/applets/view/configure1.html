MarvinView Example - Compression, color, magnification, atom size, hydrogen labels

In this example we have changed some parameters to configure MarvinView. The differences from the first example are enumerated below:

  • The molecule is downloaded from a compressed molfile (notice the .csmol extension).
  • The molecule has the same background color as the web page.
  • The atoms are in black and white (monochrome display).
  • The magnification of the zoom window (which appears if you click the molecule) is bigger.
  • Atom labels are bigger.
  • Hydrogens labels are shown for all the atoms.
